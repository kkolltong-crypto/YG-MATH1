# 과세특 자동 생성 도구 (수학과)

양진중학교 1학년 1학기 수학과 평가계획서를 바탕으로 수행평가 문항 설계부터 반별 성적 입력, 학교생활기록부 "세부능력 및 특기사항"(과세특) 문장 생성까지 한 화면에서 처리하는 브라우저 도구입니다. 서버나 별도 설치 없이 `index.html` 파일을 웹브라우저로 열기만 하면 바로 사용할 수 있습니다.

## 사용 방법

1. 이 저장소를 내려받거나 `git clone` 합니다.
2. `index.html` 파일을 크롬 등 웹브라우저로 엽니다.
3. 화면 위쪽 탭을 순서대로 사용합니다.

## 탭 구성

- **⓪ 평가계획서 업로드·편집**: 평가계획서 PDF를 올리면 텍스트를 추출해 참고할 수 있고, 성취기준·수행평가 목록을 표에서 직접 입력·수정할 수 있습니다. JSON으로 저장해두면 다른 과목의 평가계획서도 같은 틀로 관리할 수 있습니다.
- **① 평가계획 개요**: ⓪ 탭에서 정리한 평가 횟수·배점·시기·성취기준을 한눈에 봅니다.
- **② 문항 생성 마법사**: 성취기준 선택 → 평가요소 확인·수정 → 문항 초안 생성 → 채점기준표(부분점수) → 교사 검토·확정의 5단계로 수행평가 문항을 설계합니다.
- **③ 반별 성적 입력**: 반을 선택하면 학번이 자동으로 채워지고(반별 30번까지), 학생별로 상/중/하/미실시를 입력합니다. CSV로 내보내거나 불러올 수 있습니다.
- **④ 세특 생성기**: ③에서 입력한 성적을 불러와 반 전체 학생의 세특 문장을 자동 생성합니다. "하" 평가에는 개선 방향을 함께 붙이고, "미실시"로 표시한 단원은 문장에서 제외됩니다.

## 폴더 구조

```
.
├── index.html              # 메인 도구 (이 파일만 열면 바로 사용 가능)
├── README.md
├── docs/
│   ├── 평가계획서_분석.md     # 수학과 평가계획서 분석 결과
│   └── 요구사항명세서.md      # 최초 기능 요구사항 정리
└── samples/
    └── 수학과_1반_평가입력_샘플.csv   # 입력 형식 예시
```

## 참고

- 현재 성취기준·평가요소·채점 문구는 양진중학교 수학과 1학년 1학기 평가계획서를 기준으로 만들어졌습니다. 다른 과목에 적용하려면 ⓪ 탭에서 성취기준/수행평가 목록을 새로 입력하거나, `docs/평가계획서_분석.md`를 참고해 유사한 구조로 데이터를 채워 넣으면 됩니다. 다만 ③·④ 탭(성적 입력·세특 생성기)은 현재 수학과의 6개 평가요소 구조에 맞춰져 있어, 다른 과목의 평가요소 개수·구성이 다르면 해당 부분 코드도 함께 수정해야 합니다.
- PDF 텍스트 추출 기능은 인터넷 연결이 필요합니다(pdf.js CDN 사용).
- 브라우저에 저장 기능(localStorage 등)을 사용하지 않으므로, 입력한 내용은 CSV/JSON으로 내보내 보관해야 합니다.
